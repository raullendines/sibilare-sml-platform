# Product Scope

## MVP Scope

- Client and user management.
- Client branding.
- Brands, subbrands, competitors and competitor subbrands.
- Platform configuration.
- Apify extraction configs, jobs and runs.
- Normalized posts.
- Topics and subproducts.
- AI classification taxonomy.
- Dashboard widgets.
- PNG and CSV export tracking.
- PDF/PPT report generation with history.
- Basic chatbot with monthly limits.
- Usage ledger for cost-bearing operations.

## Advanced Scope

- AI-assisted dashboard personalization.
- Pattern detection.
- Advanced client presentations.
- Escalations and notification routing.
- Cost budget dashboards.
- Recommendation engine for competitors or topics.
