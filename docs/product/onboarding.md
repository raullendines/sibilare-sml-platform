# Client Onboarding

Client onboarding is an operational workflow, not just a settings screen.

## Steps

1. Create client.
2. Configure branding.
3. Configure users and roles.
4. Define own brands and subbrands.
5. Define competitors and competitor subbrands.
6. Configure supported platforms.
7. Estimate volume by brand/platform.
8. Configure extraction frequency and max posts.
9. Configure topics and subproducts.
10. Customize dashboard from base widgets.
11. Configure report templates.
12. Validate first extraction.
13. Validate first dashboard.
14. Validate first report.
15. Activate client.

## Required Tracking

The onboarding process should be represented with `onboarding_projects` and `onboarding_tasks` so the managed service team can operate consistently.
